# Blog Website

A full-stack blog application where users can create, read, update, and delete blog posts with authentication functionality.

## Live Demo

https://blog-website-backend-3gpk.onrender.com

## Features

- User Signup & Login Authentication
- Create Blog Posts
- Edit & Delete Posts
- MongoDB Database Integration
- REST API Integration
- Full-stack MERN-style architecture

## Tech Stack:

### Frontend
- React.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

## Project Structure

```bash
blog-website/
```
